Picking apart JS - alert

Open Chrome Web Inspector, and go to the Console tab.

Type in the following, hitting Enter after each semi-colon:

alert;
alert("hi");
alert("goodbye");
typeof "hi";
typeof "goodbye";

alert is a function which can be called with a string.

When it is called, it displays the string to the user.

Picking apart JS - document

Open Chrome Web Inspector, and go to the Console tab.

Type in the following, hitting Enter after each semi-colon:

document;

document is an object representing our entire web page.

Expand the arrow to see what's inside.

Picking apart JS - getElementById

Open Chrome Web Inspector, and go to the Console tab.

Type in the following, hitting Enter after each semi-colon:

getElementById;
document.getElementById;
document.getElementById("goodbye");
document.getElementById("message");
document.getElementById("message").value;

getElementById is a function attached to the document object, which can be called a string.

When it is called, it tries to find and return an element in the document with the same id as the string.

Picking apart JS - mailer and send

Open Chrome Web Inspector, and go to the Console tab.

Type in the following, hitting Enter after each semi-colon:

mailer;
mailer.send;

mailer is an object which we imported by adding this to the page:

js/mailer.js is included in your starter kit - you can open it from your editor.

send is an function attached to the mailer object, which can be called with a string.

function createFavorite() {
    // The other stuff that was here

    var newFavoriteTitle = document.getElementById("creator-title").value;
    var newFavoriteDescription = document.getElementById("creator-description").value;
    var newFavoriteMedia = document.getElementById("creator-media").value;

    // Put our form values in the new favorite
    theNewFavorite.getElementsByClassName("title")[0].innerHTML = newFavoriteTitle;
    theNewFavorite.getElementsByClassName("description")[0].innerHTML = newFavoriteDescription;
    theNewFavorite.getElementsByClassName("media")[0].innerHTML = newFavoriteMedia;
